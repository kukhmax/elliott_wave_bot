Необходимо с помощью python через API крипто бирж определить пятиволновую структуру по всем првмлам Элиота . Т.е. первая волна это либо импулс, либо начальная диагональ, вторая - коррекция ( ABC, или двойной зигзаг), третяя - импулс пятиволновой, четвертая - коррекция ( ABC, треугольник или сложная коррекция) , пятая - импульс , либо конечная диагональ. Вторая и четвертая - чередуются или два похожих флага. Все волны соответствуют пропорциям. 
Также соблюдены все правила: начало второй волны не заходит за начало первой; начало пятой волны не накладывается на конец первой. Также соответствует всем поавилам по Фибоначчи. 
Не зависимо от тренда: лонг и шорт.  И чтобы результат в виде графика приходил в телеграм бота


телеграм бот :
 кнопки :
  1. Старт для запуска бота - которая меняестя на Стоп после запуска
  2. Список пар , которые отслеживаются 
  3. Добавить пару /Изменить таймфрейм / Удалить пару - три кнопки соответственно для добавления, изменения таймфрейма и удаления пар в ручную для отслеживания
  4. Проверить пару - кнопка для проверки пары (при нажатии вводим пару и таймфрейм) на наличие пятиволновой структуры
  5. Настройки - кнопка для открытия меню настроек (настраиваем интервал для сканирования пар на наличие пятиволновой структуры, и таймфрейм для проверки пар)

логика приблизительно такая:
- При нажатии на Старт для запуска бота, бот через api coinmarketcap получает список 20 пар наивысшим marketcap (за исключением стейблкоинов , которые привязаны к фиатным валютам) добавляются в список пар, которые отслеживаются
- При нажатии на Стоп, бот останавливается и удаляет все данные о парах, которые отслеживаются
- При нажатии на Добавить пару, добавляет пару в список пар, которые отслеживаются
- При нажатии на Изменить таймфрейм, позволяет ввести новый таймфрейм для конкретной пары, и изменяет его в списке пар, которые отслеживаются
- При нажатии на Удалить пару, показывает список пар, которые отслеживаются, и позволяет выбрать которую удалить. После этого, удаляет пару из списка пар, которые отслеживаются
- При нажатии на Проверить пару, позволяет ввести пару и таймфрейм, и проверяет, есть ли в этой паре пятиволновая структура
- При нажатии на Настройки, открывает меню настроек, где можно настроить интервал для сканирования пар на наличие пятиволновой структуры

- После добавления пар, бот начинает сканировать их последние 150 свечей на наличие пятиволновой структуры
- После проверки каждой пары , если находит структуру, согласно правилам и фибоначчи, то отправляет в телеграм бот графики с изображенными волнами  и описанием структуры
- Если не находит, то не отправляет сообщение ,что совпадений не найдено
